# 📚 Native Course App — EdTech Platform

A **full-stack mobile application** built with **React Native (Expo)** that enables real-time online classes between teachers and students. Features include live video conferencing via **LiveKit**, biometric authentication, attendance tracking, class management, and AI-powered session summaries.

---

## 🚀 Features

### 👨‍🏫 Teacher Side
- Signup & Login with email/password + biometric authentication (Face, Fingerprint, Voice)
- Create and manage classes
- Start live video meetings (camera + microphone via LiveKit)
- View & manage participants in real time (kick students)
- Mark and check student attendance
- Record and view class recordings
- Dashboard with class overview

### 👩‍🎓 Student Side
- Signup & Login with email/password + biometric authentication
- Join live video meetings
- View enrolled classes
- Check personal attendance records
- View AI-generated session summaries
- Access class recordings
- Real-time notifications

### 🔐 Authentication
- Email & Password based login/signup
- Biometric multi-factor authentication:
  - Fingerprint (via device native prompt)
  - Face recognition
  - Voice recognition
- Form state persists across biometric flows (no input clearing)
- Secure JWT-based sessions via backend API

### 📹 Live Video — LiveKit Integration
- Real-time video conferencing using [LiveKit Cloud](https://livekit.io/)
- Camera and microphone tracks published via `@livekit/react-native`
- Teacher controls: mute/unmute camera & mic
- Student viewing: live video feeds from teacher
- Participant presence detection (join/leave events via Socket.IO)
- Auto-kick students removed by teacher from the active session
- Token-based room authentication (token generated by backend)

## ⚙️ Setup & Installation

### Prerequisites
- Node.js (v18+)
- Expo CLI: `npm install -g expo-cli`
- Android Studio / Android device with USB debugging
- Backend server running (Node.js + MongoDB)
- LiveKit Cloud account

### 1. Clone the Repository
```bash
git clone https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
cd YOUR_REPO_NAME
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Configure API & LiveKit URL

Open `config.js` and update:

```js
const IP_ADDRESS = 'YOUR_LOCAL_IP'; // e.g., 192.168.1.11

export const BASE_URL = `http://${IP_ADDRESS}:5000/api`;
export const SOCKET_URL = `http://${IP_ADDRESS}:5000`;
export const PYTHON_SERVICE_URL = `http://${IP_ADDRESS}:5001`;
export const LIVEKIT_URL = 'https://YOUR_LIVEKIT_CLOUD_URL.livekit.cloud';
```

> ⚠️ **Note:** Update `IP_ADDRESS` every time your PC restarts and gets a new local IP. Run `ipconfig` (Windows) to find it.

### 4. Run the App

```bash
# Start Expo dev server
npx expo start

# Run on Android
npx expo run:android

# Run on iOS
npx expo run:ios
```

## 📹 LiveKit Setup Guide

LiveKit powers the real-time video/audio in this app. Here's how it's wired up:

### Step 1 — Create a LiveKit Cloud Account
1. Go to [https://livekit.io/](https://livekit.io/) and sign up (free tier available)
2. Create a new **Project**
3. Copy your:
   - **WebSocket URL** (e.g., `wss://your-project.livekit.cloud`)
   - **API Key**
   - **API Secret**

### Step 2 — Backend: Generate LiveKit Token
Your Node.js backend must generate a LiveKit room **token** for each user. Install the LiveKit server SDK:

```bash
npm install livekit-server-sdk
```

Example token generation route:
```js
const { AccessToken } = require('livekit-server-sdk');

app.post('/api/livekit/token', async (req, res) => {
  const { roomName, participantName } = req.body;

  const token = new AccessToken(
    process.env.LIVEKIT_API_KEY,
    process.env.LIVEKIT_API_SECRET,
    { identity: participantName }
  );

  token.addGrant({
    roomJoin: true,
    room: roomName,
    canPublish: true,
    canSubscribe: true,
  });

  res.json({ token: await token.toJwt() });
});
```

Add to your `.env`:
```
LIVEKIT_API_KEY=your_api_key
LIVEKIT_API_SECRET=your_api_secret
```

### Step 3 — Frontend: Connect to LiveKit Room

The app uses `@livekit/react-native` hooks:

```js
import { useRoom, useLocalParticipant } from '@livekit/react-native';

const { connect, room } = useRoom();

// Fetch token from your backend, then connect:
await connect(LIVEKIT_URL, token, { audio: true, video: true });
```

### Step 4 — Android Permissions
Already added in `app.json`:
```json
"permissions": [
  "android.permission.CAMERA",
  "android.permission.RECORD_AUDIO",
  "android.permission.MODIFY_AUDIO_SETTINGS",
  "android.permission.ACCESS_NETWORK_STATE"
]
```
